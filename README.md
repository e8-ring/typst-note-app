### リンク機能

概要 : 各メモに一意的な識別子を割り当てる (数字 & 大文字アルファベット 4 桁)。その値を取得できるようにする。メモ A の識別子値を所定の方法でメモ B 内に記述するとリンクが貼られ、メモ B からメモ A へ飛べるようにする。

その他アイデア :

- リンクをクリックしたときに、中身をその場で展開するように設定することもできるようにしたい。
